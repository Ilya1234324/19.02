Краткий отчёт о работе с Git
git init — «запустил» Git в папке «Задания 19.02» (репозиторий уже существовал).
git add . — подготовил все файлы в папке для сохранения (точка — «все файлы»).
Ошибка с git git commit — опечатка: дважды написал git, Git не понял команду.
git commit -m "Initial commit" — создал первый «снимок» (коммит) файлов.
git branch -M main — переименовал ветку с master на main.
git remote add origin https://... — попытался связать папку с GitHub, но получил ошибку: связь уже есть (с репозиторием 18.02.2026).
git remote -v — проверил, с чем связан репозиторий: оказалось, с 18.02.2026.
git remote set-url origin https://... — поменял ссылку на новый репозиторий (19.02) на GitHub.
git push origin main — отправил файлы на GitHub в репозиторий 19.02 (всё сжалось и улетело!).
git remote add new-origin https://... — добавил ещё одну «связь» с GitHub (дополнительный репозиторий).
git push -u origin main — закрепил связь между локальной веткой main и GitHub, настроил «автоматический» push.
